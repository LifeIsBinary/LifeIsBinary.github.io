window 使用的tip

发表于2022-12-05|更新于2022-12-05|电脑使用tips
|字数总计:185|阅读时长:1分钟|阅读量:

快捷键

  1. win+E：打开资源管理器
  2. win+D：隐藏/显示所有桌面上界面
  3. win+L：锁屏
  4. win+shift+s：截图
  5. win+Tab：在打开的应用程序间切换
  6. Alt + 双击左键：查看文件属性
  7. win +. :调出window 下的表情和公式符

应用窗口快速缩小放大

Alt + 空格 + x ：全屏

Alt + 空格 + r ：还原

Alt + 空格 + n：缩小

通过鼠标双击应用窗口的标题栏,也可以实现全屏和窗口还原操作。

鼠标左键拖动文件到另一地方时：

  • 按住Shift并释放为移动文件操作
  • 按住Ctrl并释放为文件复制

实用软件

  1. 空格键查看文件内容——quicklook
  2. 快速文件检索——everything
